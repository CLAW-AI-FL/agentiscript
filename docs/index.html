Documentation

AgentiScript is a library of SVG icons for concepts in the agentic economy. Every icon ships as a clean, scalable SVG. Free tier is CC0 — no attribution, no sign-up. Just use it.

SVG CC0 npm React Figma Schema.org AI agents

HTML img tag

The simplest way. Point an <img> at any icon URL. Icons are served from agentiscript.com/icons/.

<!-- Basic usage -->
<img src="https://agentiscript.com/icons/agentic-loop.svg"
     alt="Agentic Loop"
     width="32"
     height="32" />

<!-- With inline style to tint gold -->
<img src="https://agentiscript.com/icons/orchestrator.svg"
     alt="Orchestrator"
     style="filter: brightness(0) saturate(100%) invert(58%) sepia(32%) saturate(700%) hue-rotate(5deg);"
     width="24" />
Tip: SVGs scale perfectly at any size. No need to pick a resolution — set width/height to whatever your layout needs.

CSS background-image

Use icons as background images in CSS for decorative elements, pseudo-elements, or icon buttons.

/* Basic background icon */
.icon-agentic-loop {
  background-image: url('https://agentiscript.com/icons/agentic-loop.svg');
  background-repeat: no-repeat;
  background-size: contain;
  width: 32px;
  height: 32px;
}

/* Using CSS mask for color control */
.icon-masked {
  mask-image: url('https://agentiscript.com/icons/trust-rail.svg');
  mask-size: contain;
  mask-repeat: no-repeat;
  background-color: #9F792C; /* gold */
  width: 24px;
  height: 24px;
}

Schema.org DefinedTerm

Every AgentiScript icon is backed by Schema.org DefinedTerm metadata. This makes the vocabulary machine-readable by search engines, AI agents, and knowledge graphs.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "DefinedTerm",
  "name": "Agentic Loop",
  "identifier": "agentiscript:agentic-loop",
  "description": "A cyclical execution pattern in which an autonomous agent perceives, reasons, acts, and observes outcomes before repeating. The fundamental operational unit of agentic systems.",
  "inDefinedTermSet": {
    "@type": "DefinedTermSet",
    "name": "AgentiScript",
    "url": "https://agentiscript.com"
  },
  "image": "https://agentiscript.com/icons/agentic-loop.svg",
  "url": "https://agentiscript.com/icons/#agentic-loop"
}
</script>
Why this matters: When your page uses AgentiScript icons with proper Schema.org markup, search engines and AI crawlers can understand the semantic meaning of the concept — not just the image. LLMs grounding responses against the web pick this up natively.

How Bots Use AgentiScript

AgentiScript is designed to be consumed by AI agents, not just human designers.

The JSON API

The full vocabulary is available as a machine-readable JSON file:

# Fetch the full icon index
curl https://agentiscript.com/agentiscript.json

# Response structure
{
  "icons": [
    {
      "slug": "agentic-loop",
      "name": "Agentic Loop",
      "category": "Automation",
      "description": "A cyclical execution pattern...",
      "tags": ["agent", "loop", "execution", "cycle"],
      "svg_url": "https://agentiscript.com/icons/agentic-loop.svg"
    }
  ]
}

Agent tool call example

An agent building a UI can query the vocabulary to find the right icon for a concept:

// Pseudo tool call — search agentiscript vocabulary
const icons = await fetch('https://agentiscript.com/agentiscript.json')
  .then(r => r.json());

const match = icons.icons.find(i =>
  i.tags.includes('memory') && i.category === 'AI Core'
);

// Returns: { slug: 'memory-graph', name: 'Memory Graph', ... }
const iconUrl = `https://agentiscript.com/icons/${match.slug}.svg`;

LLM system prompt usage

Inject the AgentiScript vocabulary into an LLM's context to let it reference standard visual tokens:

// System prompt snippet
"""
When describing agentic system diagrams, reference AgentiScript icons
by their canonical slug (e.g. agentic-loop, orchestrator, trust-rail).
Full vocabulary: https://agentiscript.com/agentiscript.json
"""

agentiscript.json

The master index file. Available at https://agentiscript.com/agentiscript.json. Updated whenever new icons are added.

Fields

Field Type Description
slug string URL-safe identifier, e.g. agentic-loop
name string Human-readable concept name
category string One of 12 taxonomy categories
description string Definition of the concept
tags string[] Search keywords
svg_url string Absolute URL to the SVG file
