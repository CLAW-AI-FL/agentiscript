# AgentiScript

**The visual language of the agentic economy.**

3,790 emoji exist. Zero represent the agentic economy.

AgentiScript fills that gap.

Every concept the agentic economy speaks but cannot show gets a visual token.

## What is in here

- /icons/ SVG icons + JSON metadata for every concept
- agentiscript.json machine-readable index of all icons

## Use it

<img src="https://agentiscript.com/icons/agentic-loop.svg" />

npm install agentiscript
